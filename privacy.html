1. 政策總覽

「研究生讀 Paper 神器」是一款協助使用者匯入 PDF 或圖片、擷取文字、整理閱讀筆記、建立引用資料，並在使用者選擇啟用 AI 功能時進行摘要、翻譯、分類與研究整理的應用程式。

我們重視你的隱私。本 App 的設計原則是以裝置本地保存為主，並只在你主動使用相關功能時處理必要資料。

2. 我們處理的資料

2.1 你提供或匯入的內容

  • 你匯入的 PDF、圖片或其擷取後文字內容
  • 你建立或編輯的標題、作者、標籤、關鍵字、研究筆記、引用資料
  • 你輸入的 OpenAI API Key 與 AI 模型設定

2.2 使用過程中產生的資料

  • 文字擷取、OCR、摘要、翻譯、分類、研究方法、貢獻與實驗設計等分析結果
  • App 內部保存所需的本地資料，例如索引資訊、匯入檔案的保存位置與資料庫內容

2.3 我們不主動蒐集的資料

  • 本 App 不要求建立帳號
  • 本 App 不主動蒐集姓名、電話、住址等個人身分資料，除非你在來信聯繫時自行提供
  • 本 App 不提供站內廣告，也不建立廣告追蹤檔案

3. 資料如何使用

我們處理資料的目的包括：

  • 讓你匯入、閱讀、整理與搜尋 Paper
  • 在裝置本地保存你的 Paper、筆記、標籤與引用內容
  • 在你主動啟用 AI 分析時，產生摘要、翻譯、分類與研究整理內容
  • 回覆你寄給開發者的支援需求或問題回報

4. 第三方服務

4.1 OpenAI

若你在 App 中填入自己的 OpenAI API Key 並使用 AI 功能，App 會將你選定文件中的部分文字內容傳送至 OpenAI，以完成摘要、翻譯、分類與研究分析。

這些資料的處理也可能受 OpenAI 的服務條款與隱私政策約束。建議你在啟用前自行確認相關政策是否符合你的需求。

4.2 GitHub Pages

本隱私權政策頁面可能透過 GitHub Pages 提供靜態網站託管。若你瀏覽本頁，GitHub 可能依其平台規則處理基本技術性請求紀錄。

5. 保存、控制與刪除

  • Paper 內容、筆記、標籤與分析結果主要保存在你的裝置本地。
  • 你可以在 App 內刪除單篇 Paper，相關本地副本也會一併移除。
  • 你也可以透過移除 App 來刪除裝置上的本地資料。
  • 若你曾使用 AI 功能，傳送到 OpenAI 的內容不由本 App 直接控制保存週期，需參考其服務政策。

6. 使用者權利與選擇

  • 你可以選擇不填入 OpenAI API Key，此時仍可使用匯入、整理與本地保存功能。
  • 你可以自行編輯或刪除大部分 Paper 資料與引用欄位。
  • 若你對資料處理方式有疑問，可透過下方聯繫方式與開發者聯絡。

7. 兒少隱私

本 App 並非以 13 歲以下兒童為主要對象設計。我們不會明知而蒐集兒童個人資料。如你認為相關資料被不當提供，請透過電子郵件與我們聯繫。

8. 政策更新

我們可能會因 App 功能調整、法規要求或服務流程變更而更新本政策。更新後的版本會發布在本頁面，並以頁面上標示的生效日期為準。
